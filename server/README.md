# Member App - Server

This is a minimal Express backend for the Member App.

Quick start:

1. Copy `.env.example` to `.env` and set `MONGO_URI` to your MongoDB Atlas connection string.
2. Install deps: `npm install` inside `server`.
3. Run in dev: `npm run dev` (requires `nodemon`).
